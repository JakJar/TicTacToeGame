# Tic-Tac-Toe Game

### About:
Tic-Tac-Toe is a very simple two player game.
This game is also known as Noughts and Crosses or Xs and Os game. 
One player plays with X and the other player plays with O. 
In this game we have a board consisting of a 3X3 grid.
To win players must place their characters in one line - vertically, horizontally or diagonally.
